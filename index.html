Artifact 3: Clustering Analysis with K-Means

Title: Customer Segmentation with K-Means Clustering

Objective: Apply unsupervised learning techniques to segment data, identify hidden patterns, and visualize clusters.

Process: Data preprocessing, determining optimal clusters using Elbow and Silhouette methods, applying K-Means, and visualizing clusters.

Tools & Technologies: Python (Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn), Jupyter Notebook.

Key Results: 3 optimal clusters identified with a silhouette score of 0.72.
