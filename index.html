Artifact 3: Clustering Analysis with K-Means

Title: Customer Segmentation with K-Means Clustering

Objective: Apply unsupervised learning techniques to segment data, identify hidden patterns, and visualize clusters.

Process: Data preprocessing, determining optimal clusters using Elbow and Silhouette methods, applying K-Means, and visualizing clusters.

Tools & Technologies: Python (Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn), Jupyter Notebook.

Value Proposition: Demonstrates mastery of unsupervised learning and data visualization, enabling actionable insights from unlabeled data.

Key Results: Identified 3 optimal clusters with a silhouette score of 0.72.

Class Activity: Training Methods Coach

In this AIML-500 activity, I explored key training methods such as supervised, unsupervised, and reinforcement learning. I learned the differences in how models learn from data and apply feedback loops for performance improvement.

Reflection

This artifact demonstrates my ability to apply unsupervised learning techniques and visualize meaningful insights from unlabeled data. I chose this project because clustering models are crucial for customer segmentation, and it complements my supervised learning knowledge. I incorporated peer feedback by adding detailed graphs (Elbow Method and Cluster Visualization) and documenting the process thoroughly.

The 3.3 Training Methods Coach activity reinforced my understanding of algorithm selection and training workflows, which I applied while optimizing K-Means parameters. By integrating these lessons, I improved the clarity and accuracy of my results. This artifact not only highlights technical proficiency but also demonstrates the ability to tailor my work for professional and academic audiences.
