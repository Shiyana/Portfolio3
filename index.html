Artifact 3: Clustering Analysis with K-Means

Title: Customer Segmentation with K-Means Clustering

Objective: To apply unsupervised learning techniques for data segmentation, discovering hidden patterns, and visualizing clusters.

Process: Data preprocessing (cleaning, scaling), determining optimal clusters using Elbow and Silhouette methods, applying K-Means, and visualizing results with 2D scatter plots.

Tools & Technologies: Python (Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn), Jupyter Notebook.

Value Proposition: This artifact demonstrates expertise in unsupervised machine learning and the ability to uncover actionable insights from unlabeled data.

Key Results: Identified 3 optimal clusters with a silhouette score of 0.72, delivering clear, interpretable groupings.

Reflection

I selected this artifact because it showcases a unique machine learning skill—unsupervised clustering—that complements my earlier work on supervised models. This project helped me understand how to identify patterns without labeled data and reinforced my data visualization skills. I refined the artifact for a professional audience by including the Elbow Method and Silhouette visualizations, making the results more interpretable.
