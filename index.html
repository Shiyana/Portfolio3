Artifact 4: Machine Learning Scenarios and Ethical Reflection

Shiyana Jayanesan Shylaja | Machine Learning Portfolio

Overview

This artifact captures my engagement with three real-world machine learning problem scenarios guided by a chatbot, complemented by a leadership reflection on human bias in AI and education. Through structured exercises, I tackled missing data, noisy sensor readings, and class imbalance. Each scenario included decision-making processes, strategies, and coding examples.

These were paired with ethical reflections on leadership in the AI space, where I explored inclusive design, data audits, and stakeholder involvement. The integration of technical analysis and reflective writing provided a holistic foundation in practical and principled machine learning practices.

Process

Step 1: Analyzed missing data patterns and applied imputation techniques using Pandas and SimpleImputer.
Step 2: Cleaned noisy sensor signals using moving average and statistical outlier removal.
Step 3: Used SMOTE and class weighting to manage class imbalance in rare disease prediction.
Step 4: Documented reflections and applied learnings on ethical leadership.

Chatbot-Guided ML Scenarios

Scenario 1 – Missing Data: Explored strategies such as median imputation, predictive modeling, and missingness flags using Python.

Scenario 2 – Noisy Data: Applied visualization, smoothing techniques, and rolling averages to clean data and improve signal clarity.

Scenario 3 – Imbalanced Data: Utilized SMOTE and model evaluation metrics (F1-score, ROC-AUC) to counter bias in rare event classification.

Key Takeaways

  • Handled real-world data challenges in a structured and repeatable way.
  • Learned to select the right preprocessing techniques based on data quality issues.
  • Understood the ethical responsibilities tied to AI leadership.
  • Recognized the value of reflection in reinforcing technical learning.

Reflection

This assignment challenged me to think like both a machine learning engineer and a responsible leader. By working through three progressive data issues, I solidified my knowledge of preprocessing and evaluation methods while learning to justify decisions and handle uncertainty.

Reflecting on AI and bias through the lens of education reinforced my belief that technical mastery is incomplete without ethical grounding. I now approach AI projects with greater empathy and intention.

Conclusion

This artifact strengthened both my practical and reflective capacities in AI. I plan to carry these strategies forward into future roles where both accuracy and accountability matter. I also aim to keep exploring bias mitigation techniques and stakeholder feedback models in real-world deployments.
