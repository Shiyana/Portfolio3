Artifact 3: Clustering Analysis with K-Means

Overview

This artifact demonstrates the use of K-Means clustering to segment customer data into meaningful groups. The purpose of this project is to identify hidden patterns within datasets for better decision-making.

Objective: Use unsupervised learning to cluster data, identify trends, and visualize clusters effectively for business insights.

Tools & Technologies: Python, Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn, Jupyter Notebook.

Process

Step 1: Data Preprocessing
Cleaned and normalized the dataset to ensure uniform scaling, handled missing values, and prepared features for clustering.

Step 2: Optimal Cluster Identification
Applied the Elbow Method and Silhouette Score to identify the optimal number of clusters (3 in this case).

Step 3: K-Means Algorithm
Implemented K-Means clustering, assigned labels, and computed metrics to evaluate cluster quality.

Step 4: Visualization
Generated scatter plots to visualize clusters and centroids, ensuring interpretability.

Challenges & Solutions

  • Challenge: Determining the correct number of clusters was initially ambiguous.
    Solution: Used a combination of Elbow Method and Silhouette Score for accurate selection.
  • Challenge: Handling unscaled features distorted clustering results.
    Solution: Applied feature scaling (StandardScaler) to standardize the dataset.

Results & Key Takeaways

The model successfully identified 3 clusters with a silhouette score of 0.72, indicating good separation between the clusters. The visualizations provided actionable insights into customer behavior.

Key Takeaways:

  • Mastered unsupervised learning concepts and the practical implementation of K-Means.
  • Improved understanding of data preprocessing and the impact of feature scaling.
  • Learned to use metrics and visualization tools to evaluate clustering quality.
