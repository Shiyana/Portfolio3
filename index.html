Chatbot-Guided ML Scenarios

Scenario 1: Missing Data

Problem: Missing patient data (age, weight) in a health dataset.

Solution: Imputed missing values using median and regression. Flagged missingness. Explored impact using data visualization.

from sklearn.impute import SimpleImputer
imputer = SimpleImputer(strategy='median')
df[['age', 'weight']] = imputer.fit_transform(df[['age', 'weight']])

Reflection: Learned to balance completeness with bias prevention.

Scenario 2: Noisy Data

Problem: Sensor data with spikes and fluctuations distorted trends.

Solution: Used IQR-based outlier removal, rolling means, and exponential smoothing to denoise.

df['smoothed'] = df['reading'].rolling(window=5, center=True).mean()

Reflection: Gained confidence in tuning smoothing methods without losing critical info.

Scenario 3: Imbalanced Dataset

Problem: Disease classification with very few positive examples (5%).

Solution: Used SMOTE to oversample and class-weighted models to train with AUC and F1 evaluation.

from imblearn.over_sampling import SMOTE
smote = SMOTE()
X_res, y_res = smote.fit_resample(X, y)

Reflection: Learned the importance of using precision-recall over accuracy in medical AI tasks.
